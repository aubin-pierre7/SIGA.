# SIGA — Système Intégré de Gestion d'Archives

Projet de fin de DUT Informatique.

## Description
SIGA est une application web sécurisée de gestion et d'archivage
de documents administratifs. Il intègre le chiffrement AES-256,
un système d'authentification par rôles, un journal d'audit complet,
et des modules d'intelligence artificielle pour la classification
automatique des documents.

## Stack technique
- **Backend** : FastAPI (Python)
- **Frontend** : React.js + Tailwind CSS
- **Base de données** : SQLite (dev) → PostgreSQL (prod)
- **Chiffrement** : AES-256-GCM
- **IA** : scikit-learn + spaCy
- **Auth** : JWT
